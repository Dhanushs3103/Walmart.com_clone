# Walmart.com Clone

### Description
Walmart.com Clone is a replica of the popular retail website Walmart.com, offering a wide range of products including groceries, electronics, fashion apparel, and home goods at competitive prices.

## Tech Stack Used:
  - React.js
  - Chakra UI
  - Chakra Icons
  - React Icons
  - React Router DOM

## Functionality
  - Slider Bar
  - Product Listings
